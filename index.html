Como jogar:

  1. Peppa (rosinha) começa o jogo
  2. Clique em uma peça para selecionar e depois em uma casa vazia para mover
  3. Peças comuns movem-se apenas para frente na diagonal
  4. Para capturar, pule sobre a peça do oponente para uma casa vazia
  5. Quando uma peça chega ao outro lado, vira uma Dama (com coroa) e pode mover para trás
  6. O jogo termina quando um jogador não tem mais peças ou não pode mover

Regras do Jogo de Damas

  1. O tabuleiro tem 8x8 casas com 12 peças para cada jogador
  2. Peças comuns movem-se apenas para frente na diagonal
  3. Capturas são feitas pulando sobre a peça adversária
  4. Se uma captura for possível, ela deve ser feita
  5. Quando uma peça alcança a última linha, torna-se uma Dama
  6. Damas podem mover e capturar para frente e para trás
  7. O jogo termina quando um jogador perde todas as peças ou não pode mover
